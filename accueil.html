C'est quoi Github?

GitHub est une plateforme web et un service basé sur le cloud qui utilise le système
de contrôle de version Git pour faciliter la collaboration des développeurs sur des
projets de logiciels.
Elle permet de stocker du code, de suivre les modifications au fil du temps,
de gérer le travail d'équipe et de partager des projets, en particulier ceux de type
« open source ». GitHub est utilisé pour tout, des projets logiciels aux listes de contenu,
et il héberge des millions de projets et de développeurs à travers le monde.

Fonctionnalités principales

Contrôle de version avec Git : GitHub intègre Git, un système décentralisé qui permet
de sauvegarder des versions de projets, de suivre les modifications et de revenir à
des versions antérieures.

Collaboration en équipe : C'est une plateforme idéale pour les équipes, qu'elles soient
à distance, afin de travailler ensemble sur le même code,
de partager leur travail et de gérer les contributions.

Hébergement de projets : Les projets sont stockés dans des « dépôts »
(repositories), qui peuvent être publics pour
une visibilité maximale ou privés pour des projets d'entreprise.

Partage et découverte : Il est facile de parcourir les projets existants, de faire
des copies modifiables (fork) pour y contribuer et d'épingler des projets pour les suivre
ou les mettre en avant.
